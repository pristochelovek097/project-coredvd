# Project coreDVD

это проект с реальными прошивками с собственным ассемблером и процессором, потом появятся и сторонние процессоры.

структура репозитория:

```bash
project-coredvd
├── LICENSE
├── README.md
├── docs.md
├── firmware
│   └── Pulse
│       ├── docs.md
│       └── firmware.asm
└── src
    └── main.cpp
```

мы назвали наш процессор pulse.
